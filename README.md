# frontend-tareas

Con este proyecto, puedes ver como crear una app basica para gestionar tareas pendientes con vue.js .

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
